# APEX CORE SETUP

In Apex, the biggest trigger for bans is manual bans.

&#x20;I recommend playing with spectators enabled.&#x20;

Try not to get too many kills per match.

&#x20;Don't exceed 8 kills per game. Almost no one gets more than that in this game; it's extremely rare to get many kills per match because the game usually has long distances and slow pace.

&#x20;Don't overuse obvious aimbots

Try to hide your walls.
