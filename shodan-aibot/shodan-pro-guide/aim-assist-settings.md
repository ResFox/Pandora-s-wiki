#### Right Column: <a href="#right-column" id="right-column"></a>

* Master Speed:
  * Controls how fast you reach the target, 0 is the slowest, 1 is the fastest. Test this as needed.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2Fm3rKU35u0sDNGc9fiY5j%2Fimage.png?alt=media&amp;token=e653216a-f292-4109-8566-a6728e3ad8b0" alt=""><figcaption></figcaption></figure>

\
Aim Speed X/Y:

* Controls how fast, individually on each axis, you reach the target. This value always aligns with the Master Speed unless you set it otherwise.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2F94tdUAMDcjNCWmQ1jzFs%2Fimage.png?alt=media&amp;token=dad7d7b4-9e34-4379-abee-7a96a86bbdb4" alt=""><figcaption></figcaption></figure>

\
Flick Speed:

* Adjusts how quickly aim snaps to targets. Higher values create faster, more aggressive movement. Lower values make it slower to reach the target.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2Fqd2YocfE1yJweWfaWTT1%2Fimage.png?alt=media&amp;token=af7b6f6f-a145-460b-8fd3-8b463f6c6ef3" alt=""><figcaption></figcaption></figure>

\
Overshoot Correction:

* Controls stability when approaching targets. Higher values prevent aim from oscillating or overshooting. Basically, if you experience your mouse going in circles, turn this up.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2FwhmMqNuQ7jnTjAYYkVFc%2Fimage.png?alt=media&amp;token=df374dd2-bdcf-4b7e-9ba6-5375125bb149" alt=""><figcaption></figcaption></figure>

\
Sensitivity Scale:

* Changes the speed of the overall aim system. This value will most likely need to be changed, as many people differ in DPI and in-game sens. (**If your mouse isn't moving, this value is most likely what needs to be changed!!!**)

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2FzEdTRy0HmuTllwQ91oa8%2Fimage.png?alt=media&amp;token=f13444a1-dd68-491f-aec2-e6b8066c1bae" alt=""><figcaption></figcaption></figure>

\
Confidence %:

* Changes the threshold at how confident the model has to be in order for it to aim. If you set this too low, it'll give false detections, if you set it too high, it may not detect enemies anymore. Test as needed. Depends on the model.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2F1qWc9jtaa20RB6ChWwYQ%2Fimage.png?alt=media&amp;token=27f2cf4b-6a24-41fa-bfc5-34be5aafca31" alt=""><figcaption></figcaption></figure>

Aim Delay:

how much software takes to start aiming at target. Ideal for legit users&#x20;

#### Triggerbot Settings: <a href="#triggerbot-settings" id="triggerbot-settings"></a>

* Triggerbot checkbox:
  * Turns on and off the automatic Triggerbot.

<figure><img src="https://2000999588-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fxr02iQbBt5IsdQ3g6zQo%2Fuploads%2FGxBnQJ1Z5lGmmjeSPPZX%2Fimage.png?alt=media&amp;token=3f2623db-31b5-4584-aa7e-dbbd11cebd4c" alt=""><figcaption></figcaption></figure>

\
Triggerbot Mode:

* One-Shot:
  * Shoots once when it gets to the center point of the target.
* Spray:
  * Shoots multiple times when it gets to the center point of the target.

\
Trigger on Key Checkbox:

* If checked, the triggerbot only activates when the specified key is held down.
