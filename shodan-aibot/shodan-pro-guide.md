{% embed url="general-set-up/pc-enviroment.md" %}

1. Extract files using file password.&#x20;

{% hint style="info" %}
Pls do not use file inside the zipped folder .
{% endhint %}

2. Run Shodan as **Admin** and Wait for **auth widow popup.**
3. Put your license and click Enter.
4. When Shodan open for the first time i recommend close and then go to Models discord channel and download models for your favorite game.
5. Put the models inside the /models folder
6. Open Shodan and go to poison tab and select your favorite game model.
7. Enjoy playing

{% hint style="info" %}
note: is posible your shodan config is not perfect, can be really slow or so shaky depends of your ingame sens, dpi and windows config.&#x20;
{% endhint %}
