HOW TO NOW IF YOUR CONTROLLER AIMBOT WILL WORK ?

1. Disconect your real controller.
2. Open Shodan
3. Select Controller as Driver
4. And Follow this

Press `WINDOWS KEY + R`

Write **joy.cpl** and press enter

in Joy.cpl you should see only 1 controller

<figure><img src="https://2000999588-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252Fxr02iQbBt5IsdQ3g6zQo%252Fuploads%252FsW9tmQbywwYqfIrauCu4%252Fimage.png%3Falt%3Dmedia%26token%3D8ce83a7d-b977-4618-a989-a1c59a13f6f9" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
This is the virtual controller Shodan creates for you. If you have your controller disconected and you see more than 1 controller here, maybe is related to other periferial, some Razer, Corsair or other gaming devices are recognized as Game Controller by windows. You have to disconect them all. You can plug back after correctly setup Controller index.
{% endhint %}
