# 🖱️ Aim Not Working

First make sure you have installed the RawAccel component. We use it to auto-setup the mouse acceleration for you. The software searches for it at start, and if it is not present, aim just never starts.

You can find it on the download website.

{% hint style="warning" %}
### I had RawAccel installed but aim does not work

In this case you need to follow this:

1. Uninstall RawAccel.
2. Restart your PC.
3. Install the RawAccel version provided by Pandora's Box.
4. Restart your PC.
5. Disconnect your mouse from the PC and connect it again (plug and unplug).
6. Open the RawAccel software.
7. In RawAccel, go to the Advanced / Device menu.
8. Make sure your mouse is selected and not a different device.
9. Some gaming keyboards and gamepads confuse RawAccel.
{% endhint %}
