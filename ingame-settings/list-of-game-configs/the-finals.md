# The Finals

1- Open windows search and enter %LOCALAPPDATA%\
2- Find Discovery\Saved\Config\WindowsClient folder\
3- Use Notepad to open GameUserSettings.ini\
4- Set the FrameRateLimit to 60 or 144. Depeneds of your screen refresh rate
